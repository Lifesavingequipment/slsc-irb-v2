## 2. TYPOGRAPHY

### Font Family

```css
font-family: "Inter", ui-sans-serif, system-ui, sans-serif;
```

Loaded via `@fontsource/inter` (or CDN). Applied globally on `body`.

### Heading Tracking

All headings (`h1`, `h2`, `h3`) use:

```css
letter-spacing: -0.02em;
```

### Size Scale

| Tailwind Class | Approx Size | Usage |
|----------------|------------|-------|
| `text-xs` | 12px | Badges, timestamps, captions, small labels |
| `text-sm` | 14px | Body text, card descriptions, button labels, nav items |
| `text-base` | 16px | Input/select default text |
| `text-lg` | 18px | Section headings |
| `text-xl` | 20px | Sub-page headings |
| `text-2xl` | 24px | Page titles (h1) |
| `text-7xl` | 72px | 404 error display |

### Font Weights

| Class | Weight | Usage |
|-------|--------|-------|
| `font-normal` | 400 | Base body text |
| `font-medium` | 500 | Labels, secondary headings |
| `font-semibold` | 600 | Card titles, emphasis text |
| `font-bold` | 700 | Page titles, primary headings |

### Common Text Patterns

```
Page title:      text-2xl font-bold
Page subtitle:   text-sm text-muted-foreground
Card title:      font-semibold leading-none tracking-tight
Card description: text-sm text-muted-foreground
Badge text:      text-xs font-semibold
Nav label:       text-sm font-medium
Mobile nav label: text-xs font-medium
Small label:     text-[10px] uppercase tracking-wider
Notification badge: text-[10px] or text-[9px] font-bold
```

## 3. COMPONENTS

### Button (`src/components/ui/button.tsx`)

Base classes: `inline-flex items-center justify-center gap-2 whitespace-nowrap rounded-md text-sm font-medium transition-colors cursor-pointer`

Focus: `focus-visible:outline-none focus-visible:ring-1 focus-visible:ring-ring`

Disabled: `disabled:pointer-events-none disabled:opacity-50 disabled:cursor-not-allowed`

SVG children: `[&_svg]:size-4 [&_svg]:shrink-0`

#### Variants

| Variant | Classes |
|---------|---------|
| `default` | `bg-primary text-primary-foreground shadow hover:bg-primary/90` |
| `destructive` | `bg-destructive text-destructive-foreground shadow-sm hover:bg-destructive/90` |
| `outline` | `border border-input bg-background shadow-sm hover:bg-accent hover:text-accent-foreground` |
| `secondary` | `bg-secondary text-secondary-foreground shadow-sm hover:bg-secondary/80` |
| `ghost` | `hover:bg-accent hover:text-accent-foreground` |
| `link` | `text-primary underline-offset-4 hover:underline` |

#### Sizes

| Size | Classes |
|------|---------|
| `default` | `h-9 px-4 py-2` |
| `sm` | `h-8 rounded-md px-3 text-xs` |
| `lg` | `h-10 rounded-md px-8` |
| `icon` | `h-9 w-9` |

```jsx
    <div className="flex-1 min-w-0">{/* content */}</div>
    {isExpanded ? <ChevronDown className="h-4 w-4" /> : <ChevronRight className="h-4 w-4" />}
  </button>
  {isExpanded && <div className="mt-3 pt-3 border-t">{/* expanded content */}</div>}
</Card>
```
